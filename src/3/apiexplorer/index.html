    Schedule

    GET /schedule.json

    Returns a schedule for the specified channel and period.

    Parameter Value Type Description
    channel_id Please Select cbbh (BBC One) cbbG (BBC Two) cbbP (BBC Three) cbbQ (BBC Four) cbbV (BBC HD) cbbW (CBBC) cbcQ (BBC Radio 1) cbcS (BBC Radio 2) string The id of the required channel. Available from /channels.json
    from multiple The beginning of the schedule period. Can be an ISO date, number of seconds since 1 Jan 1970 or a relative duration, e.g. 'now'.
    to multiple The end of the schedule period. Can be an ISO date, number of seconds since 1 Jan 1970 or a relative duration., e.g. 'now.plus.6h'.
    publisher string The publisher of the schedule
    annotations list List of annotations that describe the data to be output
    Valid Annotations: description, extended_description, brand_summary, series_summary, people, topics, segment_events, related_links, key_phrases, broadcasts, locations, first_broadcast, next_broadcast, available_locations, plus channel.

    Request

    Response

    • JSON

    • Other available format(s):

    • XML

    Channels and Channel groups

    GET /channels.json

    Returns a paginated list of all channels.

    Parameter Value Type Description
    broadcaster string Optionally filter by the broadcaster that owns the channel
    media_type string Optionally filter by the type of channel (audio or video)
    available_from string Optionally filter by a publisher that publishes data for the channel
    order_by string Optionally order the list of channels
    limit integer The number of channels to return per page
    offset integer The offset from the start of the list.
    annotations list List of annotations that describe the data to be output
    Valid annotations: channel_groups, history, parent, variations

    Request

    Response

    • JSON

    • Other available format(s):

    • XML

    Search

    GET /search.json

    Returns a list of content that matches the specified search criteria.

    Parameter Value Type Description
    q string The title of a container or item
    limit integer The number of items to return (max 50).
    publisher string The publisher of the content.
    Valid annotations: none

    Request

    Response

    • JSON

    • Other available format(s):

    • XML

    Response

    • JSON

    • Other available format(s):

    • XML
    TitleSearch for brand, series and one-off pieces of content. Can't be used with genre
    PublisherYour application will be restricted to certain publishers. You can further restrict the results to specific publishers.
    GenreThe genre of the content you would like.
    TagYour application will be restricted to search for certain publishers. You can further restrict the results to specific publishers.
    AvailableTrue or false. 'any' will give you content that is either available or not.
    Available CountriesCountry code eg. uk, us etc.
    DurationLength of the content
    Is Long FormTrue or false. Only full length content?
    Data Container FormatThe format of the content eg. mp4, mp3 etc.
    Transport TypeHow you get to the content. 'link' or 'embed' are common examples.
    Media TypeEither 'audio' or 'video'.
    FromThe beginning of a schedule period.
    ToThe end of a schedule period.
    ChannelThe channel that the content was broadcast on eg. bbcone, bbctwo etc.
    copyClick to copy the query to your clipboard
    brandA special type of container that can only contain episodes
    SpecialA one off item that does not have a container
